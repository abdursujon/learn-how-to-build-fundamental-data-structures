# Learn How to Build Fundamental Data Structures From Scratch 

This repo covers a practical approach to building must-know data structures. 
As developers, we should know how to build our own tools, not just how to use them.
This repo helps us understand how things work under the hood and how we can build data 
structures our own way, adding things we wish some data structures had.
